# Unassuming

An RSpec formatter to take up minimal space

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'unassuming'
```

And then execute:

    $ bundle

## Usage

add to `.rspec`

```ruby
--format Unassuming
```
